## Description

webide的后端项目，数据库用的是mongo，数据库配置在ormconfig.json

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
